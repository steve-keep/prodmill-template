# ProdMill Template

This is a template repository for creating new ProdMill projects.

## Configuration

To use the ProdMill integration, you need to configure the `JULES_API_KEY` secret in your repository.

1. Go to your repository's **Settings**.
2. In the left sidebar, click **Secrets and variables**, then click **Actions**.
3. Click **New repository secret**.
4. Enter `JULES_API_KEY` as the name of the secret.
5. Enter your Jules API key as the value of the secret.
6. Click **Add secret**.
